glyphicons.less

A less file to ease the use of the GLYPHICONS.

Download this project as a .zip file Download this project as a tar.gz file

Usage

Add glyphicons.less to your project and add this to your less file

@iconSize: 20px;
@iconPath: "png/";

@import "glyphicons.less";

Adding icons is simple just add this wherever you want an icon. The code below will give you the icon named random.

<i class="icon-random"></i>

The icons is not included, get them at GLYPHICONS.com

About GLYPHICONS

GLYPHICONS is a library of precisely prepared monochromatic icons and symbols, created with an emphasis on simplicity and easy orientation. More info about the icons here: GLYPHICONS.com
